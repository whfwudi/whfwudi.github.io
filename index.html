OSX10.11更新cocoapods和使用

发表于 2016-01-08   |   分类于 cocoapods   |  

cocoapods的安装

替换ruby更新源

`gem sources --remove https://rubygems.org/ 
 gem sources -a https://ruby.taobao.org/`

注意是https，不是http 现在的http不好用了

验证更新是否成功

`gem sources -l`

如果出现下面代码则是成功了

`*** CURRENT SOURCES ***  
 https://ruby.taobao.org/`

更新ruby环境

`sudo gem update --system`

安装cocoapods

`sudo gem install cocoapods`

如果遇到错误：

`ERROR:While executing gem ... (Errno::EPERM)
 Operation not permitted - /usr/bin/pod`

可以用下面的命令进行修改：

`sudo gem install -n /usr/local/bin cocoapods`

安装成功后，执行下面命令

`pod search mknet`

如果出现commond not find.可以查看/usr/local/bin/下的pod文件，通过编辑~/.bash_profile,添加

`export PATH=/usr/local/bin:$PATH`

这样就是可以正常使用cocoapods了

pod的使用命令

打开终端，在终端中输入pod，点击回车，就会出现pod的一系列命令。

安装好cocoapods知识必然要用一下，满足一下自己的好奇心啊，接下来就开始介绍cocoapods的使用过程

cocoapods的使用

新建xcode工程

然后输入下列命令

`pod search mknet`

结果如图

cd到xcode工程的总目录下

输入下列命令

`touch Podfile`
 vim Podfile
 platform :ios, '7.0' 
 pod 'MKNetworkKit', '~> 0.87'`

然后按esc，并输入:wq,回车之后会发现工程目录下多了一个Podfile文件
最后输入下列命令：

`pod install`

等待几分钟之后就会提示打开**.xcworkspace，这样我们的第三方库就安装成功了。

在同一个工程中再次加入其他第三方库

如果一个工程中已经通过cocoapods安装过一次第三方库，再次用cocoapods导入第三方库的时候就不需要上面那么繁琐的过程了只需要下列步骤就可以了：

  1. 执行搜索库的命令

    `pod search mbprogress`

    复制

    `pod 'MBProgressHUD', '~> 0.9.2'`

  2. 打开工程中的Podfile文件

    粘贴复制好的代码到工程中

  3. 执行下列命令，再次等待安装完成就可以了

    `pod install`
  4. 打开工程目录下的后缀为xcworkspace的文件，我们会发现工程中多了我们需要的第三方库，恭喜你，你已经学会使用cocoapods了，多加使用就会提高使用的熟练度。

iOS缓存清理功能的实现

发表于 2016-01-07   |   分类于 iOS   |  

移动端应用在处理网络资源的时候，一般会做一些离线缓存的处理，其中最为典型的就是图片缓存，然而图片缓存是最不好处理的，其中非常流行的第三方离线缓存框架为SDWegImage。但是，离线缓存的内容太多又会占用手机的存储空间，所以缓存清理功能成为咨询，阅读等类app的标配功能。所以今天就来介绍一下离线缓存的清理功能，功能分为两步：

获取缓存文件的大小

由于缓存文件都是存在沙盒中，所以我们可以通过NSFileManager API来实现计算缓存文件大小的功能

  1. 计算单个文件的大小

    `(float)fileSizeAtPath:(NSString *)path{
        NSFileManager *fileManager=[NSFileManager defaultManager];
    if([fileManager fileExistsAtPath:path]){
    long long size=[fileManager attributesOfItemAtPath:path error:nil].fileSize;
    return size/1024.0/1024.0;
    }
    return 0;
}`

  2. 计算目录大小

    `+(float)folderSizeAtPath:(NSString *)path{
    NSFileManager *fileManager=[NSFileManager defaultManager];
    float folderSize;
    if ([fileManager fileExistsAtPath:path]) {
    NSArray *childerFiles=[fileManager subpathsAtPath:path];
    for (NSString *fileName in childerFiles) {
        NSString *absolutePath=[path stringByAppendingPathComponent:fileName];
        folderSize +=[FileService fileSizeAtPath:absolutePath];
    }
    folderSize+=[[SDImageCache sharedImageCache] getSize]/1024.0/1024.0;
    return folderSize;
        }
    return 0;    
}`

清除缓存文件

同样也是利用NSFileManager API来实现，SDWebImage框架自己实现了清理缓存操作，我们可以直接调用。

`+(void)clearCache:(NSString *)path{
    NSFileManager *fileManager=[NSFileManager defaultManager];
    if ([fileManager fileExistsAtPath:path]) {
        NSArray *childerFiles=[fileManager subpathsAtPath:path];
        for (NSString *fileName in childerFiles) {
            //如有需要，加入条件，过滤掉不想删除的文件
            NSString *absolutePath=[path stringByAppendingPathComponent:fileName];
            [fileManager removeItemAtPath:absolutePath error:nil];
        }
    }
    [[SDImageCache sharedImageCache] cleanDisk];
}`
